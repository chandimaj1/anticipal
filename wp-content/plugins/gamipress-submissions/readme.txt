Allow users to unlock achievements and ranks through admin reviewed submissions.

== Description ==

Submissions lets you bring to your users the ability to unlock achievements and rank through a form submission that your website administrators can review to accept or reject it.

In just a few minutes, you will be able to place submissions forms around your site where your users will be able to request the unlock of any achievement and rank.

In addition, this add-on includes the ability to enable notes with the submission form where users can explain why they should earn it and an administrative screen to manage all submissions received.

= Features =

* Enable achievements or ranks of your choice for admin reviewed submissions.
* Optionally, you can enable a notes field to let users provide an explanation with the submission.
* Settings to configure the submission form texts.
* Administrative screen to approve or reject any submission.
* Ability to revoke already approved submissions and restore the user awards.
* Admin emails to get notified about any new submission.
* Blocks and Shortcodes to place any submission form anywhere.
* Widgets to place any submission form on any sidebar.
* New fields and attributes on GamiPress blocks, shortcodes and widgets to manually show or hide the configured submission settings.

== Installation ==

= From WordPress backend =

1. Navigate to Plugins -> Add new.
2. Click the button "Upload Plugin" next to "Add plugins" title.
3. Upload the downloaded zip file and activate it.

= Direct upload =

1. Upload the downloaded zip file into your `wp-content/plugins/` folder.
2. Unzip the uploaded zip file.
3. Navigate to Plugins menu on your WordPress admin area.
4. Activate this plugin.
